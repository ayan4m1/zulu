# Zulu Framework

## Setup

Install the dependencies:

```bash
npm install
```

## Get Started

Start the dev server:

```bash
npm run dev
```

Build the app for production:

```bash
npm run build
```

Serve built app for production:

```bash
npm run serve
```

Build and serve built app for production:

```bash
npm start
```
